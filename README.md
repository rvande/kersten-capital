## On-demand revalidation (Strapi webhook)

To keep blog and homepage fresh when you publish in Strapi without short ISR intervals:

1. In Vercel (and `.env.local`), set `REVALIDATE_SECRET` to a random string (e.g. from `openssl rand -hex 24`).
2. In Strapi: **Settings → Webhooks → Create**. URL: `https://your-domain.com/api/revalidate`, Method: **POST**. Add header: `x-revalidate-token` = your `REVALIDATE_SECRET`.
3. Trigger the webhook on blog post create/update/publish so the site revalidates on demand.
